# Turtle, run!
How to play?
```
W : UP
S : DOWN
A : LEFT
D : RIGHT
K : Dig
L : Dig
```
How to run?
```
$ cd cantonese
$ python cantonese.py ../game.cantonese 
```
Enjoy it!
